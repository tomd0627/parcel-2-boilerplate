This is a static Front-End boilerplate which uses Parcel.js to quickly get set up with some common frameworks and plugins.

  • Bootstrap 5
  • Font Awesome 6 Free
  • jQuery
  • PostHTML Include (for including HTML partials)
  • SASS
  • Select2 (for custom styling of select elements)
  • Slick Carousel (accessible, WCAG 2.0 / 2.1 compliant)

Note: You will need node v10 or higher.

  1. Run npm install or yarn install
  2. Run npm start or yarn start to start a development server with LiveReload.
  3. Run npm run build or yarn build to build all files and assets for production in a dist folder.

Note: Each file with the name index is a master import file.
